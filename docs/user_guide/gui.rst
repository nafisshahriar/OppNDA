GUI Guide
=========

This guide covers using the OppNDA graphical interface.

Starting the GUI
----------------

.. code-block:: bash

    python OppNDA.py

Open http://localhost:5000 in your browser.

Main Pages
----------

Dashboard (nda.html)
~~~~~~~~~~~~~~~~~~~~

The main dashboard showing simulation results and visualizations.

Settings (settings.html)
~~~~~~~~~~~~~~~~~~~~~~~~

Configure simulation parameters and post-processing options.

Features
--------

- **Configuration Management** - Edit simulation and post-processing settings
- **Real-time Output** - View process output as it runs
- **Post-processing Controls** - Run averager, analysis, or regression individually
